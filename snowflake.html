Snowflake Tutorial ❄️

Learn Snowflake concepts with examples

What is Snowflake?

Snowflake is a cloud-based data warehouse platform that separates compute and storage for better scalability and performance.

Snowflake Architecture

  • Cloud Storage Layer
  • Compute Layer (Virtual Warehouses)
  • Cloud Services Layer

Create Table Example

CREATE TABLE EMPLOYEE (
  ID INT,
  NAME STRING,
  SALARY NUMBER
);

Insert Data

INSERT INTO EMPLOYEE VALUES (1, 'Kiran', 50000);
INSERT INTO EMPLOYEE VALUES (2, 'John', 60000);

Query Data

SELECT * FROM EMPLOYEE;

Time Travel Feature

Snowflake allows you to access historical data using Time Travel. 

SELECT * FROM EMPLOYEE
AT (TIMESTAMP => '2024-01-01 10:00:00');

Load Data using COPY INTO

COPY INTO EMPLOYEE
FROM @my_stage/file.csv
FILE_FORMAT = (TYPE = 'CSV');
